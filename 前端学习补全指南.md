# 前端学习补全指南

基于模拟面试结果，按优先级从高到低整理学习路线。

---

## P0：必须掌握（面试高频）

### 1. 浏览器渲染原理

**学习目标**：能完整描述从输入 URL 到页面渲染的全过程

**知识点清单**：
- [ ] DNS 解析流程（递归查询 + 缓存层级）
- [ ] TCP 三次握手 / TLS 握手
- [ ] HTTP 请求响应流程
- [ ] HTML 解析 → DOM 树构建
- [ ] CSS 解析 → CSSOM 树构建
- [ ] DOM + CSSOM → 渲染树
- [ ] 布局（Layout/Reflow）→ 绘制（Paint）→ 合成（Composite）
- [ ] JS 阻塞 DOM 解析、async/defer 区别

**推荐资料**：
- 文章：《从输入 URL 到页面展示，这中间发生了什么？》— 极客时间浏览器工作原理
- 文章：https://web.dev/howbrowserswork/
- 视频：李兵《浏览器工作原理与实践》— 极客时间

### 2. 浏览器缓存策略

**学习目标**：能完整描述强缓存和协商缓存的判断流程，准确说出相关 Header

**知识点清单**：
- [ ] 强缓存：Cache-Control（max-age、no-cache、no-store、public、private、immutable）
- [ ] 强缓存：Expires（已过时）
- [ ] 协商缓存：Last-Modified / If-Modified-Since
- [ ] 协商缓存：ETag / If-None-Match
- [ ] 304 Not Modified 的含义
- [ ] 完整的缓存判断流程图
- [ ] 带 hash 的静态文件缓存策略 vs HTML 入口缓存策略

**练习**：在 Chrome DevTools Network 面板中观察实际请求的缓存 Header

**推荐资料**：
- 文章：https://web.dev/http-cache/
- MDN：HTTP Caching

### 3. CORS 跨域原理

**学习目标**：能区分简单请求和预检请求，理解完整的 CORS 工作流程

**知识点清单**：
- [ ] 同源策略的定义（协议 + 域名 + 端口）
- [ ] 简单请求的条件和流程
- [ ] 预检请求（OPTIONS）的触发条件和流程
- [ ] 关键 Header：Access-Control-Allow-Origin/Methods/Headers/Credentials
- [ ] 其他跨域方案：JSONP、Nginx 反代、postMessage、WebSocket

**练习**：用 Node.js 搭建一个简单的 CORS 服务器，用不同方式触发预检请求

**推荐资料**：
- MDN：Cross-Origin Resource Sharing
- 文章：https://web.dev/cross-origin-resource-sharing/

## 学习建议

1. **优先级排序**：先攻克 P0（1-2 周），再攻克 P1（2-3 周），P2 持续积累
2. **学习方法**：每个知识点做到"能讲清楚"而不是"看过"，尝试用自己的话讲出来
3. **源码阅读**：Vue 3 的 reactivity 模块源码量不大，非常值得直接阅读
4. **推荐书籍**：
   - 《Vue.js 设计与实现》（霍春阳）— 必读，覆盖 Vue 3 核心原理
   - 《深入浅出 Node.js》— 网络相关
   - 《JavaScript 高级程序设计》（第 4 版）— JS 基础
5. **每日练习**：每天手写 1 道题 + 复习 1 个知识点，面试前完成所有清单
